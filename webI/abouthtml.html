O que é HTML?

O HTML (HyperText Markup Language) é a linguagem fundamental para a criação de páginas web, fornecendo a estrutura necessária para exibir conteúdo na internet. Desenvolvido por Tim Berners-Lee em 1991, o HTML evoluiu ao longo dos anos, com a versão mais recente sendo o HTML5, que introduziu recursos avançados para suportar multimídia, gráficos vetoriais e APIs para geolocalização, entre outros.
