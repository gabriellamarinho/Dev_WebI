O que é CSS?

CSS, ou Folhas de Estilo em Cascata, é uma linguagem fundamental no desenvolvimento web, responsável por definir a aparência e o layout de páginas HTML. Enquanto o HTML estrutura o conteúdo da página, o CSS determina como esse conteúdo será apresentado ao usuário, incluindo aspectos como cores, fontes, espaçamento e posicionamento dos elementos.

Estrutura Básica do CSS

O CSS é composto por regras que consistem em seletores e declarações. Um seletor identifica o(s) elemento(s) HTML a serem estilizados, e a declaração especifica o estilo a ser aplicado. Uma declaração é composta por uma propriedade e um valor.

            
    seletor {
      propriedade: valor;
    }

Por exemplo, para alterar a cor do texto de todos os parágrafos para azul, utilizamos:

            
    p {
      color: blue;
    }

Formas de Aplicar CSS

Existem três maneiras principais de aplicar CSS a um documento HTML:

  1. CSS Externo: Consiste em criar um arquivo separado com extensão .css e vinculá-lo ao documento HTML. Essa abordagem permite a reutilização de estilos em várias páginas e facilita a manutenção do código. Para vincular o CSS externo, adiciona-se a seguinte linha dentro da tag <head> do HTML: 
                        
    <link rel="stylesheet" href="estilos.css">
  2. CSS Interno: Os estilos são definidos dentro da própria página HTML, utilizando a tag <style> dentro da tag <head>. Essa abordagem é útil para páginas únicas ou quando se deseja aplicar estilos específicos a um único documento. 
                        
    <style>
      p {
        color: blue;
      }
    </style>
  3. CSS Inline: Os estilos são aplicados diretamente aos elementos HTML através do atributo style. Essa abordagem é menos recomendada, pois pode tornar o código mais difícil de manter, especialmente em projetos maiores. 
                        
    <p style="color: blue;">Texto em azul</p>

Propriedades Comuns no CSS

Além disso, o CSS oferece uma ampla gama de propriedades para estilizar elementos, como:

  • Cor de fundo: background-color
  • Fonte: font-family, font-size, font-weight
  • Espaçamento: margin, padding
  • Bordas: border
  • Alinhamento: text-align, vertical-align
